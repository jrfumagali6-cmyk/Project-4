# Project-4

Today, I am bringing a new Finance project using Python. I decided to apply PCA in Streamlit to identify clusters among the tickers and, in turn, correlated tickers. 

Moreover, I will talk a little about PCA (Principal Component Analysis); in general terms, PCA involves clustering things (e.g., tickers, skills, and other items) into common factors. In our case, it considers different types of tickers: oil, civil construction, mining, and others (for example!), So PCA will cluster these tickers into common factors. And what does this matter for us? The final remarks will indicate which groups of tickers are better to invest in, because investors sometimes invest in random or highly correlated tickers, which can be harmful to portfolio returns.
